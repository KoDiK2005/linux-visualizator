# Linux Visualizator

Десктопный виджет (десклет) для Linux, который красиво в реальном времени показывает,
как работает компьютер: загрузка CPU по ядрам, использование RAM/Swap и сетевой трафик.

## Идея

Проект разделён на два независимых слоя:

- **core** — сбор и агрегация системных метрик (не зависит от UI, читает `/proc` и использует `psutil`).
- **ui** — десклет на PySide6 (Qt): прозрачное окно без рамки, которое рисует живые индикаторы
  поверх/под рабочим столом.

Такое разделение позволяет в будущем добавить другой UI (например, веб-дашборд) без переписывания
сборщиков данных.

## Структура

```
core/
  collectors/   # cpu.py, memory.py, network.py — парсинг /proc и psutil
  models.py     # датаклассы-снапшоты метрик
  sampler.py    # периодический опрос + история для графиков
  config.py     # интервалы опроса, список интерфейсов, пороги
ui/
  widget/
    main_window.py     # frameless always-on-desktop окно
    animation.py        # плавное сглаживание значений между тиками
    panels/             # cpu_panel.py, mem_panel.py, net_panel.py
    theme.py            # цвета, прозрачность, шрифты
  app.py                # точка входа
scripts/
  install_autostart.sh  # регистрирует десклет в автозапуске (~/.config/autostart)
  uninstall_autostart.sh
tests/
```

## Статус

Этапы 0–5 из плана ниже выполнены: рабочий десклет с живой визуализацией, плавными
переходами значений, запоминанием позиции на экране и автозапуском.

## План работы

0. Скелет репозитория, зависимости, CI — **готово**.
1. Core collectors: CPU, RAM, Network + юнит-тесты парсинга `/proc` — **готово**.
2. Sampler: кольцевой буфер истории для спарклайнов, конфигурируемый интервал опроса — **готово**.
3. Базовый UI-каркас: прозрачное frameless-окно, перетаскивание, сохранение позиции — **готово**.
4. Панели визуализации: кольца загрузки CPU по ядрам, полоса RAM/Swap, спарклайн сети — **готово**.
5. Полировка: анимации значений, автозапуск, консольная точка входа — **готово**.
6. (опционально) Мини-GUI или конфиг-файл настроек.

## Запуск

```bash
python3 -m venv .venv
.venv/bin/pip install -e .
.venv/bin/python -m ui.app
```

На Debian/Ubuntu для работы Qt может понадобиться системный пакет `libxcb-cursor0`
(`sudo apt install libxcb-cursor0`).

Окно перетаскивается мышью — позиция сохраняется в
`~/.config/linux-visualizator/desklet.ini` и восстанавливается при следующем запуске.

## Автозапуск

```bash
./scripts/install_autostart.sh    # включить автозапуск при входе в систему
./scripts/uninstall_autostart.sh  # отключить
```

Скрипт создаёт `~/.config/autostart/linux-visualizator.desktop`, который подхватывается
GNOME, KDE, XFCE, Cinnamon и другими окружениями, поддерживающими XDG autostart.

## Требования

- Python 3.11+
- Linux (используется `/proc`)
- PySide6, psutil
