# Linux Visualizator

Десктопный виджет (десклет) для Linux, который красиво в реальном времени показывает,
как работает компьютер: загрузка CPU по ядрам, использование RAM/Swap и сетевой трафик.

## Идея

Проект разделён на два независимых слоя:

- **core** — сбор и агрегация системных метрик (не зависит от UI, читает `/proc` и использует `psutil`).
- **ui** — десклет на PySide6 (Qt): прозрачное окно без рамки, которое рисует живые индикаторы
  поверх/под рабочим столом.

Такое разделение позволяет в будущем добавить другой UI (например, веб-дашборд) без переписывания
сборщиков данных.

## Структура

```
core/
  collectors/   # cpu.py, memory.py, network.py — парсинг /proc и psutil
  models.py     # датаклассы-снапшоты метрик
  sampler.py    # периодический опрос + история для графиков
  config.py     # интервалы опроса, список интерфейсов, пороги
ui/
  widget/
    main_window.py     # frameless always-on-desktop окно
    panels/             # cpu_panel.py, mem_panel.py, net_panel.py
    theme.py            # цвета, прозрачность, шрифты
  app.py                # точка входа
tests/
```

## Статус

Проект в самом начале: заложен скелет архитектуры (Этап 0 плана). См. план работы ниже.

## План работы

0. Скелет репозитория, зависимости, CI — **готово**.
1. Core collectors: CPU, RAM, Network + юнит-тесты парсинга `/proc`.
2. Sampler: кольцевой буфер истории для спарклайнов, конфигурируемый интервал опроса.
3. Базовый UI-каркас: прозрачное frameless-окно, перетаскивание, сохранение позиции.
4. Панели визуализации: кольца загрузки CPU по ядрам, полоса RAM/Swap, спарклайн сети.
5. Полировка: тема/цвета, анимации значений, автозапуск (systemd user service / autostart).
6. (опционально) Мини-GUI или конфиг-файл настроек.

## Запуск (после реализации Этапа 3+)

```bash
pip install -e .
python -m ui.app
```

## Требования

- Python 3.11+
- Linux (используется `/proc`)
- PySide6, psutil
